## Governance Notes

All skills in this area operate under the **four-eyes principle**: the AI agent prepares a recommendation; a human compliance officer makes the binding decision. The agent is never authorised to issue a final onboarding disposition.

High-risk and critical skills require sign-off from the Group Risk and Compliance Committee before status is set to `approved`.
