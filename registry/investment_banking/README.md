## Governance Notes

All skills in this area operate under strict information barrier controls. The agent must never access or process material non-public information (MNPI). Any detection of MNPI triggers an unconditional veto and immediate escalation to IB Compliance.

Skills must not progress to `approved` status until the IB Compliance and Controls team has confirmed that information barrier and restricted list controls are functioning correctly in the target environment.
